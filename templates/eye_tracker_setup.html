Step-by-Step Installation Guide

1

Download & Install Tobii Driver

This driver allows your computer to communicate with the eye tracker.

1. Download the Driver:

  • Go to: gaming.tobii.com/getstarted
  • Under "1. Select the hardware", choose "Tobii" (Eye Tracking)
  • Under "2. Select Tobii device", choose "Tobii Eye Tracker 5"
  • Click the blue "DOWNLOAD DRIVER" button

Select "Tobii Eye Tracker 5" and click "DOWNLOAD DRIVER"

2. Install the Driver:

  • Find the downloaded file: Tobii.EyeTracker5.Offline.Installer_4.183.0.30025
  • Right-click the file
  • Select "Run as administrator" (important!)
  • Follow the installation prompts, click "Next" until finished
  • The installer may also install Tobii Experience Software

Tobii Experience Software installing ‚Äî wait for completion

After installation: Check Apps & Features ‚Üí search "Tobii" to verify the driver is installed, then Restart your computer.
2

Connect the Hardware

1. Plug In:

  • Connect Tobii Eye Tracker 5 to a USB port on your computer

2. Check Lights:

  • You should see faint red LEDs on the tracker when connected

3. Position:

  • Stick the magnetic mount to the bottom center of your screen
  • Attach the tracker to the mount or use double sided tape
  • Keep plugged in for all remaining steps
3

Setup Tobii Experience App

This app calibrates your eyes for tracking:

1. Open the App:

  • Click the Start Button
  • Type: Tobii Experience
  • Open the app
  • If not found: Open Microsoft Store, search "Tobii Experience", and install it

2. Create Profile & Calibrate:

  • App will ask you to create a user profile (enter name)
  • Complete Display Setup ‚Äî align the lines to the marks on top of your tracker
  • Run Calibration:
    • Follow dots on screen with EYES ONLY
    • Keep head still
    • Look at each dot until it disappears

Display Setup ‚Äî align lines to marks on your tracker

Tobii Experience app ‚Äî calibration & settings

3. Test Tracking:

  • Look around your screen
  • You should see a "bubble" or indicator following your gaze
  • If tracking is poor, recalibrate

The gaze bubble indicator follows your eyes around the screen

About Tobii Experience: This is the official app for Tobii Eye Tracker 5. Use it to improve calibration, recalibrate whenever needed, change profiles, and re-run Display Setup if you move the tracker. Recalibrate if accuracy drops or user position changes.
4

Verify Installation

After installation, check that all required software is installed:

  • Open Settings ‚Üí Apps & Features
  • Search for Tobii
  • You should see 3 apps:
    • Tobii Experience (the app from Microsoft Store)
    • Tobii Experience Software For Windows
    • Tobii Experience Software For Windows (EyeTracker5) ‚Äî the driver

Apps & Features should show these 3 Tobii components after successful installation

5

Install OptiKey

OptiKey is the actual software used for communication.

‚ö†Ô∏è CRITICAL: Do NOT download Version 4. It will NOT work with Tobii Eye Tracker 5.
You MUST use Version 3 (OptiKey Classic).

1. Download:

  • Go to: optikey.org/help/versions
  • Scroll down to find the "OptiKey Classic (V3)" section
  • Click: "Download OptiKey Pro Classic (v3.2.5)"

OptiKey versions page ‚Äî download from "OptiKey Classic (V3)" section, NOT V4!

2. Run the Installer:

  • Find the downloaded file: OptiKeySetup-3.2.5-Pro.exe
  • Right-click ‚Üí "Run as administrator"
  • The Setup Wizard will start

OptiKey Pro Setup Wizard ‚Äî click "Next" to continue

3. Select Eye Tracker:

  • When prompted, select "Tobii Eye Tracker 5" from the dropdown
  • This configures OptiKey to work with your device

Select "Tobii Eye Tracker 5" from the eye tracker dropdown

4. Complete Installation:

  • Click "Install" to begin the installation
  • Wait for the installation to complete

Click "Install" to complete the OptiKey Pro installation

6

Use OptiKey for Eye Tracker 5

1. Open OptiKey:

  • Launch the program from the Start Menu
  • The OptiKey keyboard will open on your screen

2. Test & Verify:

  • The keyboard should show "Pointing: Tobii Eye Tracker 5" in the info panel
  • Look at keys to type ‚Äî your gaze controls the cursor
  • Try typing a message to test the eye tracking

OptiKey Pro v3.2.5.0 showing "Pointing: Tobii Eye Tracker 5" ‚Äî Setup confirmed!

OptiKey keyboard in action ‚Äî type by looking at keys

Tips & Settings:
  • Recalibrate: Click "RE-CALIBRATE" button on OptiKey keyboard (bottom-left)
  • Dwell Time: Adjust in Settings ‚Üí Pointing & Selecting
  • Language: Hindi keyboard available! Go to Settings ‚Üí Words

OptiKey Pro Classic v3.2.5 working with Tobii Eye Tracker 5

Setup Complete! You now have: Tobii Experience Driver v4.183.0.30025, Tobii Experience app, and OptiKey Pro Classic v3.2.5. Full eye tracking control of keyboard, mouse, and speech output is now available through OptiKey.

  • If no Tobii software or driver found, proceed to Step 2
  • If found, uninstall ALL entries:

    • Tobii Eye Tracker 5 Drivers

    • Any other Tobii entries

  • Restart after uninstalling all Tobii software
  • This ensures all services are fully stopped

After installation completes, verify the driver was installed:

4

Configure Display & Calibrate

  • Launch Tobii Experience from Start Menu (auto-installed from Microsoft Store)
  • Position the Eye Tracker 5 at the bottom center of your monitor
  • Complete Display Setup to align the tracker position
  • Position yourself within the tracking zone (45-95 cm from screen)
  • Run Calibration ‚Äî follow the dots with your eyes
Tip: Allow the Eye Tracker to position correctly on your face. A positioning guide will appear showing where your eyes are in the tracking zone.
5

Enable Windows Eye Control

Navigate to Windows Eye Control Settings:

  • Open Settings ‚Üí Ease of Access (or Accessibility in Windows 11)
  • Select Eye Control from the left menu (under "Vision" section)
  • Toggle "Turn on eye control (EN-US keyboard layout only)" to ON
  • RESTART your computer (critical step!)
  • After restart, Eye Control should recognize the tracker
  • The Eye Control launchpad (toolbar) will appear at the top of screen

Windows Settings ‚Üí Ease of Access ‚Üí Eye Control with toggle ON

Important: You MUST restart your computer after enabling Eye Control. The feature may not work properly without a restart.
Known Issues with Beta Driver:
  • May crash on startup with newer Windows builds
  • Compatibility issues with some Windows updates
  • No official support from Tobii ‚Äî driver is discontinued

If you experience problems: Switch to Tab 1 (OptiKey) for a stable experience.
